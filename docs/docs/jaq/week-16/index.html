    Function del(path_expression) #

    The builtin function del removes a key and its corresponding value from an object.

    Examples #

    Input

    { "foo": 42, "bar": 9001, "baz": 42 }

    Query

    jq 'del(.foo)'   # --> {"bar": 9001, "baz": 42}
