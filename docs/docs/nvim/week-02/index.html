    The ‘g’ command #

    It is quite common to wish to run some specific command on every line containing a pattern. This is exactly what the ‘g’ command is for.

    Syntax #

    The ‘g’ command is executed in command mode and the syntax is following:

    :g/pattern/command

    Also the inverse is possible (run command on lines that do not match)

    :g!/pattern/command

    or

    :v/pattern/command

    Examples #

    Delete all lines containing pattern # 

    :g/foo/d

    Delete all empty lines # 

    :g/^\s*$/d

    Move imports to start of file # 

    :g/import/m0

    Capitalize all lines not containig bar # 

    :v/bar/normal gUU
