    Special character classes #

    Vim has some builtin character classes that might come in handy. Mostly useful for regex magic, but can be also used elsewhere.

    Character classes #

    • \f - Filename characters
    • \i - Identifier characters
    • \k - Keyword characters
    • \p - Printable characters
