    Numbered registers #

    • "0 has the last yank
    • "1 has the last delete (d, c, s, x)
    • "2..9 have previous deletes
    Side note
    Note that %, (), /, n and N (and maybe some more) dont got to delete registers.

    Small delete register #

    This register contains the deletes that delete less than one line. This is also somewhat confusing register as some commands don’t put stuff there while others do. Don’t have any good usecase, but never know :D

    Named registers #

    • Store whatever you want
    • Macros are stored here automatically
    • You can edit macros by editing contents of these registers

    Edit macro a with:

    let @a='<macro>'

    Use <C-r>a to access previous macro from "a register in insert mode.
